## Overview

AI-Powered Retail Analytics Dashboard is a cloud-native analytics platform built using Google Cloud Platform, BigQuery, Streamlit, Plotly, Docker, and Grok AI.

The platform enables businesses and analysts to:

* Analyze retail sales data from Google BigQuery
* Upload CSV and Excel files for instant analytics
* Generate interactive visualizations
* Obtain AI-powered business insights using Grok AI
* Access the application through a publicly deployed cloud environment

This project demonstrates end-to-end data engineering, cloud deployment, business intelligence, and AI integration capabilities.
